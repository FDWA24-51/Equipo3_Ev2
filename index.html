Evidencia 2 - CONTENIDO DINÁMICO UTILIZANDO JAVASCRIPT

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Es uno de los lenguajes más utilizados en el desarrollo web junto con HTML y CSS.

Características de JavaScript

  • Lenguaje interpretado: Se ejecuta directamente en el navegador sin necesidad de compilación.
  • Orientado a objetos: Aunque no es puramente orientado a objetos, JavaScript permite trabajar con objetos y prototipos.
  • Dinámico: No es necesario definir el tipo de datos de una variable, ya que se asigna dinámicamente en tiempo de ejecución.
  • Multiplataforma: Funciona en diferentes sistemas operativos y navegadores.

Elementos clave de JavaScript

  • Variables: Permiten almacenar información. Ejemplo: let nombre = "Juan";
  • Funciones: Conjuntos de instrucciones que realizan una tarea específica. Ejemplo: function saludar() { alert("Hola"); }
  • Eventos: Permiten responder a acciones del usuario, como clics o movimientos del ratón.
  • DOM: JavaScript puede manipular el Document Object Model (DOM) para cambiar el contenido de una página.

Aplicaciones de JavaScript

JavaScript tiene un amplio uso en el desarrollo web y otras áreas. Algunas de sus principales aplicaciones incluyen:

  • Desarrollo de páginas web dinámicas: Añadir interactividad, animaciones y validaciones de formularios.
  • Desarrollo de aplicaciones web: Junto con frameworks como React, Angular o Vue.js.
  • Aplicaciones móviles: Con herramientas como React Native o Ionic.
  • Desarrollo de juegos: Creación de juegos interactivos basados en navegador.
  • Aplicaciones del lado del servidor: Con Node.js se puede crear servidores y manejar bases de datos.

Frameworks y bibliotecas de JavaScript

Existen numerosos frameworks y bibliotecas de JavaScript que facilitan el desarrollo de aplicaciones web. Algunos de los más populares son:

  • React: Biblioteca de Facebook para crear interfaces de usuario.
  • Angular: Framework de Google para el desarrollo de aplicaciones web.
  • Vue.js: Framework progresivo para la creación de interfaces de usuario.
  • jQuery: Biblioteca que simplifica la manipulación del DOM y la interacción con el servidor.
  • Express: Framework para el desarrollo de aplicaciones web con Node.js.

Conclusión

JavaScript es un lenguaje de programación fundamental en el desarrollo web, que permite crear páginas dinámicas e interactivas. Su versatilidad y amplio uso en diferentes áreas lo convierten en una herramienta indispensable para los desarrolladores.

Ejercicios de Practica

1. Saludo al usuario

Crear un programa que solicite al usuario su nombre y muestre una alerta con el mensaje “Hola, [nombre], ¡seas bienvenido(a)!”.

2. Cálculo del costo final de un producto

Diseñar un programa que calcule el costo total de un producto en una factura.

3. Operaciones con dos números

Elaborar un programa que pida al usuario dos números (pueden incluir decimales) y realice la operación seleccionada.

4. Tipo de triángulo

Crear un programa que determine el tipo de triángulo según la longitud de sus lados.

5. Conversor de monedas

Diseñar un programa que permita al usuario seleccionar una moneda de origen, ingresar un monto a convertir y elegir la moneda de destino.

6. Calificación de una materia

Desarrollar un programa que calcule la calificación final de una materia y determine si el usuario aprobó o reprobó.

7. Contador de clics

: Crear un programa que cuente cuántas veces el usuario hace clic en un botón.

8. Tabla de operaciones

Diseñar un programa que muestre una tabla según el tipo de operación seleccionada y la cantidad de operaciones que se desean visualizar.

9. Conversión de decimal a binario

Elaborar un programa que convierta un número decimal a su representación en binario y muestre el resultado en el espacio correspondiente.

10. Juego de adivinanza de números

Crear un programa que genere un número entero aleatorio que el usuario debe adivinar.

11. Simulación de examen

Diseñar un programa que simule un examen.

12. Validación de formulario

Crear un programa que valide la entrada de datos en un formulario utilizando expresiones regulares.
